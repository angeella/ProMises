GPASub {ProMises}R Documentation

ProMises model

Description

Perform functional alignment of a matrix by von Mises Fisher Procrustes model with known reference matrix

Usage

GPASub(X, Q = NULL, k, ref_ds, scaling = TRUE, reflection = TRUE)

Arguments

X

data, i.e., matrix with dimension time points - voxels

Q

value of the location parameter of the prior distribution. It has dimension voxels x voxels, it could be not symmetric.

k

value of the concentration parameter of the prior distribution

kQ

product between the location parameter and the concentration parameter of the prior distribution. Could be supplied instead of k and Q

ref_ds

reference matrix to align

scaling

Flag to apply scaling transformation

reflection

Flag to apply reflection transformation

Value

vMFP returns a list with two components:

Xest

the aligned matrix

R

the rotation matrix

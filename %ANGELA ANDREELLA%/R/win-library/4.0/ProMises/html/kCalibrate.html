kCalibrate {ProMises}R Documentation

Calibrate parameters

Description

Calibrates the parameters of the von Mises-Fisher distribution to be used as prior in the von Mises-Fisher-Procrustes model.

Usage

kCalibrate(D, p = 0.01)

Arguments

D

squared euclidean distance matrix between the coordinates of the voxels

p

distances to compute bandwith value. Default value is 0.01

Value

Returns directly k*Q, i.e. the product between the concentration parameter and the location matrix parameter of the prior distribution

Author(s)

Angela Andreella and Daniela Corbetta
